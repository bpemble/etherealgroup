# Ethereal Group

Corporate website for **Ethereal Group**, a privately held investment and operating holding company.

## Companies

| Entity | Domain |
|---|---|
| Ethereal Capital | Finance & Investment |
| Ethereal Properties | Real Estate |
| Ethereal Trust | Estate Planning |
| Ethereal Indemnity | Captive Insurance |
| Ethereal Management | Property Management |

## Structure

```
etherealgroup/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/           # icons, fonts, favicons
└── media/            # images, video
```

## Stack

Plain HTML, CSS, and JavaScript — no build tools or dependencies required. Open `index.html` directly in a browser or serve with any static host.

## Fonts

[Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) and [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts.
